# FPAA-GRUPO-5

Repositório dos trabalhos práticos da disciplina de **Fundamentos de Projeto e Análise de Algoritmos (FPAA)** - 2025.

## Sobre o Repositório

Este repositório contém implementações de algoritmos fundamentais desenvolvidas ao longo do curso de FPAA. Cada trabalho explora conceitos importantes de estruturas de dados, algoritmos de busca, e técnicas de análise de complexidade.

## Estrutura do Projeto

```
FPAA-GRUPO-5/
├── Trabalho 1/          # Implementação básica de algoritmos de busca em grafos
└── Trabalho 2/          # Algoritmo Flood Fill com interface interativa
```

## Trabalhos Desenvolvidos

### Trabalho 1 - Algoritmos de Busca em Grafos

Implementação de algoritmos fundamentais para navegação e busca em estruturas de grafos.

### Trabalho 2 - Flood Fill Algorithm

**Implementação completa do algoritmo Flood Fill com visualização interativa**

O Trabalho 2 é uma implementação robusta e interativa do algoritmo Flood Fill (preenchimento por inundação), usado para identificar e colorir regiões conectadas em uma matriz 2D.

#### Funcionalidades Principais:

- **Interface Gráfica Interativa**: Desenhe grids personalizados clicando para criar obstáculos
- **Animação em Tempo Real**: Visualize o preenchimento célula por célula
- **Suporte para 27+ Regiões**: Sistema de cores com paleta expandida
- **Suite Completa de Testes**: 21 casos de teste automatizados
- **Exportação de GIFs**: Gere animações do processo de preenchimento
- **Casos de Borda**: Tratamento robusto de situações especiais

#### Como Usar:

```bash
cd "Trabalho 2"
python interactive_floodfill.py
```

Consulte `Trabalho 2/V1.0.md` para documentação completa.
