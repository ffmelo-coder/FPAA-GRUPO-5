Trabalho 2 - Flood Fill V0.2

## Objetivo

Implementar o Algoritmo Flood Fill para identificar e preencher automaticamente todas as regiões conectadas em um grid 2D, usando cores/valores inteiros diferentes para cada região. Esta versão (V0.2) adiciona visualização animada e capacidade de exportar GIFs.

## O que foi adicionado nesta entrega (V0.2)

- Implementação de histórico de preenchimento para gerar frames de animação (`floodfill2.py`).
- Script de visualização e exportação de GIF (`visualize2.py`).
- Paleta de cores personalizada: branco (espaços livres), preto (obstáculos), vermelho, azul, laranja, amarelo e verde (regiões 1-5).
- Exemplos demonstrando 1, 2, 3, 4 e 5 regiões separadas (`main2.py`).
- Atualização do README base (`V0.2.md`) com instruções de execução da demo animada.

## Como gerar a animação (exemplo)

1. Instale dependências (recomendado em virtualenv):

```powershell
pip install matplotlib numpy pillow
```

2. Rode o script principal no diretório `Trabalho 2`:

```powershell
cd "Trabalho 2"
python main2.py
```

O script vai gerar múltiplos GIFs em `Trabalho 2/imgs/`:

- `exemplo_1_regiao.gif` - Demonstração com 1 região (toda vermelha)
- `exemplo_2_regioes.gif` - Demonstração com 2 regiões (vermelha e azul)
- `exemplo_3_regioes.gif` - Demonstração com 3 regiões (vermelha, azul e laranja)
- `exemplo_4_regioes.gif` - Demonstração com 4 regiões
- `exemplo_5_regioes.gif` - Demonstração com 5 regiões (todas as cores)

Você também pode executar diretamente o `visualize2.py` para um exemplo básico:

```powershell
python visualize2.py
```

## Observações técnicas

- `colorir_todas_regioes(..., record_history=True)` produz um dicionário que contém as regiões preenchidas e, sob a chave `-1`, uma lista de snapshots do grid após cada célula colorida (ordem temporal). Esse histórico é usado por `visualize2.py` para criar a animação.
- A paleta de cores é fixa:
  - **0**: Branco (espaços livres)
  - **1**: Preto (obstáculos)
  - **2**: Vermelho (primeira região)
  - **3**: Azul (segunda região)
  - **4**: Laranja (terceira região)
  - **5**: Amarelo (quarta região)
  - **6**: Verde (quinta região)
- O sistema suporta até 5 regiões distintas com cores específicas.
